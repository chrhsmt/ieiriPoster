家入一真ポスター管理
===========


必須環境
--------

- Ruby 2.0
- bundlerをインストール済みであること

インストール方法
----------------

```
> git clone https://github.com/open-election/ieiriPoster.git
> cd ieiriPoster
> bundle install --path vendor/bundle
```

動かし方
--------

- 以下のコマンドを実行

```
> SHIRASETE_API_KEY=取得したAPIキー bundle exec rails server
```
